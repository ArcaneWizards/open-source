# `@arcanewizards/midi-timecode`

[![](https://img.shields.io/npm/v/@arcanewizards/midi-timecode)](https://www.npmjs.com/package/@arcanewizards/midi-timecode)

Small library that handles generation & parsing of MIDI SMPTE timecode signals.

## Installation

```sh
pnpm add @arcanewizards/midi-timecode
```

## Requirements

This library is abstracted so that it can either be used with either
the WebMidi API, or backend (Node.js) implementations such as:

- [`midi`](https://www.npmjs.com/package/midi)
- [`@arcanewizards/midi`](https://www.npmjs.com/package/@arcanewizards/midi)
